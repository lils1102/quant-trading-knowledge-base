# 00 - 学习路线

本章回答一个最常见的问题：**小白应该按什么顺序学习量化交易？**

量化交易不是“学会 Python + 调一个模型 + 自动下单”。更准确地说，它是把金融市场、统计研究、软件工程和风险管理组合在一起，形成一套可验证、可复现、可控制的交易流程。

## 一句话总览

```text
看懂市场 -> 认识交易品种 -> 学会提出策略假设 -> 获取和清洗数据 -> 回测验证 -> 风控约束 -> 模拟盘 -> 小资金实盘 -> 监控和迭代
```

每一步都不能跳。跳过市场知识，会写出“看起来很聪明、实际不可交易”的策略；跳过数据清洗，会得到虚假的回测收益；跳过风控，会在实盘中一次错误就损失严重。

## 第 1 阶段：先知道金融市场在干什么

目标：听到“股票、债券、ETF、期货、期权、订单簿、流动性、做市商、滑点”这些词时，知道它们大概是什么意思。

你需要理解：

- 股票代表公司所有权的一部分，价格会因为公司经营、市场情绪、利率、流动性等因素波动。
- 债券更像“借钱给别人”，重点是利率、信用风险、期限和现金流。
- ETF 是交易所上市的基金，像股票一样盘中交易，但背后通常是一篮子资产。
- 期货和期权是衍生品，杠杆、到期日、保证金和非线性风险非常重要。

小白建议先从股票和 ETF 开始，不建议一开始就碰高杠杆期货、裸卖期权或复杂加密合约。

## 第 2 阶段：学会用“假设”看策略

量化策略不是神秘公式，而是一句话假设。

例子：

```text
过去 12 个月涨得多的股票，未来一段时间可能继续涨。
```

这就是动量假设。

```text
两个长期相关的资产短期价差异常扩大，未来可能回归。
```

这就是配对交易或均值回归假设。

一个合格的策略假设至少回答四个问题：

1. 为什么这个规律可能存在？
2. 谁在承担风险，谁在支付收益？
3. 交易成本会不会吃掉收益？
4. 什么情况下这个规律会失效？

如果只会说“因为回测曲线好看”，还不能算策略。

## 第 3 阶段：建立数据意识

量化研究里，很多错误都不是模型错，而是数据错。

需要特别注意：

- 复权：股票分红、拆股后价格要处理，否则收益率会错。
- 幸存者偏差：只看现在还活着的股票，会高估历史表现。
- 未来函数：用了当时还不知道的数据，回测会虚高。
- 时区和交易日历：不同市场的开盘、休市和结算时间不同。
- 数据版本：同一家公司财报可能修正，不能混用不同版本。

小白可以先用日线数据练习，不要一开始就处理 Tick 高频数据。

## 第 4 阶段：做一个诚实的回测

回测不是为了证明自己对，而是为了尽早发现策略可能哪里错。

一个基础回测至少要包含：

- 买入卖出规则
- 持仓规则
- 手续费
- 滑点
- 停牌或无法成交处理
- 样本内和样本外区分
- 最大回撤、波动率、胜率、盈亏比等指标

不建议只看年化收益率。年化收益高但最大回撤巨大、交易频率过高、换手成本不可承受，这类策略很可能不能实盘。

## 第 5 阶段：风控先于收益

实盘交易中，最重要的不是“赚得快”，而是“亏损时不失控”。

基础风控包括：

- 单笔最大亏损
- 单策略最大仓位
- 单资产最大仓位
- 总杠杆上限
- 单日最大亏损
- 最大回撤止损
- 流动性限制
- 异常行情暂停交易
- 数据中断暂停交易
- 订单失败报警

一个没有风控的盈利策略，本质上只是还没有遇到让它崩溃的市场环境。

## 第 6 阶段：再考虑实盘系统

实盘系统至少包括：

```text
行情模块 -> 策略模块 -> 风控模块 -> 订单模块 -> 券商/交易所接口 -> 持仓账户模块 -> 日志监控模块
```

工程上要做到：

- 每一笔信号能追溯来源
- 每一笔订单能查到状态
- 每一次持仓变化都有记录
- 出现异常时能自动停止或降级
- 策略版本、参数、数据版本都能复现

## 第 7 阶段：最后才是机器学习和 AI

机器学习适合做：

- 因子组合
- 非线性预测
- 文本和新闻处理
- 财报和研报摘要
- 异常检测
- 参数筛选辅助

但机器学习最容易出问题的地方是：

- 过拟合
- 标签泄漏
- 样本外失效
- 训练集和实盘环境不一致
- 模型解释不足

AI/Agent 可以帮助你读资料、写代码、整理研究假设，但不应该在没有风控和审计的情况下直接控制资金。

## 推荐学习顺序

### 第 1 个月：市场和 Python 数据分析

- 学股票、ETF、债券、期货、期权的基本概念
- 学 pandas、NumPy、matplotlib
- 能下载一只股票的数据并计算收益率、均线、波动率

### 第 2 个月：基础策略与回测

- 实现均线策略
- 实现动量策略
- 实现均值回归策略
- 加入手续费和滑点
- 学会评估最大回撤、夏普比率和换手率

### 第 3 个月：因子研究

- 学什么是因子
- 做分组回测
- 计算 IC、Rank IC
- 学会中性化、去极值、标准化
- 学会检查因子失效

### 第 4 个月：组合和风控

- 学资产配置
- 学相关性
- 学仓位管理
- 学风险预算
- 建立策略上线前检查清单

### 第 5 个月以后：实盘系统和 AI 辅助研究

- 接模拟交易接口
- 建日志和监控
- 做小资金实盘验证
- 使用 AI 辅助生成研究想法，但必须人工审查和回测

- 什么是一个可交易的策略假设？
- 如何判断回测是否可信？
- 实盘交易系统至少需要哪些模块？
- 如何避免常见的新手陷阱？
- AI 在量化中应该放在什么位置？
