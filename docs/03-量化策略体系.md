# 03 - 量化策略体系

本章目标：建立策略分类框架。看到一个策略想法时，能够判断它属于哪一类、收益来源可能是什么、需要什么数据、主要风险在哪里。

## 1. 什么是量化策略

量化策略不是“用模型预测涨跌”这么窄。更准确地说，它是一组可以被数据检验、可以被系统执行、可以被风险规则约束的交易规则。

一个完整策略至少包括：

```text
交易对象
  -> 信号来源
  -> 买入规则
  -> 卖出规则
  -> 仓位规则
  -> 风控规则
  -> 交易成本假设
  -> 失效监控
```

只写“价格突破均线就买入”还不完整。还需要知道买多少、什么时候卖、亏损怎么办、无法成交怎么办、手续费和滑点怎么算。

## 2. 趋势跟踪

趋势跟踪的核心假设是：已经形成的价格趋势，可能在一段时间内继续。

直观例子：如果一个商品价格连续几个月上涨，可能说明供需变化、资金流入或宏观因素正在推动它继续上涨。

常见信号：

- 均线突破
- 过去 N 日收益率
- 通道突破
- 动量排名
- 多周期趋势共振

优点：

- 逻辑直观。
- 适合股票、ETF、期货、外汇等多类资产。
- 在大趋势行情中可能表现较好。

风险：

- 震荡行情中容易频繁亏损。
- 入场较晚，退出也可能滞后。
- 参数过度优化会导致样本外失效。

## 3. 均值回归

均值回归的核心假设是：价格短期偏离正常水平后，未来可能回到相对合理的位置。

直观例子：一家基本面没有明显变化的公司，股价短期被情绪过度打压，之后可能反弹。

常见信号：

- 价格偏离均线
- Z-score
- RSI 等超买超卖指标
- 残差偏离
- 横截面反转

优点：

- 在震荡市场中可能有效。
- 持仓周期可以较短。
- 适合和风险控制结合。

风险：

- “便宜”可能继续更便宜。
- 趋势行情中逆势交易容易连续亏损。
- 标的发生基本面变化时，历史均值可能失效。

## 4. 套利策略

套利策略试图利用两个或多个资产之间的价格关系偏离。

常见类型：

- **配对交易**：两个长期相关的资产价差偏离后，预期重新收敛。
- **跨期套利**：同一品种不同到期月份合约之间的价差交易。
- **跨品种套利**：相关商品或相关资产之间的价差交易。
- **ETF 折溢价套利**：ETF 价格与净值之间的偏离交易。
- **统计套利**：使用统计模型寻找大量资产之间的相对定价偏差。

套利不等于无风险。常见风险包括：

- 价差继续扩大。
- 流动性突然消失。
- 融券、保证金、交易限制导致无法执行。
- 交易成本吞噬收益。
- 历史关系发生结构性变化。

## 5. 因子策略

因子策略的核心思想是：某些资产特征可能与未来收益或风险有关。

常见因子：

- 价值：估值较低的资产是否长期收益更高。
- 动量：过去表现强的资产是否短期延续强势。
- 质量：盈利能力、现金流、杠杆结构是否影响收益。
- 规模：小市值资产是否有不同风险收益特征。
- 波动率：低波动资产是否有更稳健表现。
- 流动性：流动性差的资产是否需要更高风险补偿。

因子策略常见流程：

```text
提出因子假设
  -> 构造因子值
  -> 清洗数据
  -> 分组测试
  -> IC / Rank IC 检验
  -> 中性化与风险控制
  -> 组合构建
  -> 样本外验证
```

风险：

- 因子拥挤后失效。
- 数据挖掘得到虚假因子。
- 交易成本过高。
- 行业、市值、风格暴露没有控制好。

## 6. 事件驱动策略

事件驱动策略关注特定事件对价格的影响。

常见事件：

- 财报发布
- 分红、回购、并购
- 指数成分调整
- 政策变化
- 评级调整
- 重大新闻

关键问题：

- 事件时间是否准确？
- 市场在事件前是否已经预期？
- 事件影响是短期还是长期？
- 是否有足够样本支持统计检验？
- 交易成本和冲击成本是否可接受？

事件驱动的难点在于时间戳。不能用公告发布之后才知道的信息，去模拟公告之前的交易。

## 7. 做市策略

做市策略通过持续挂买单和卖单，赚取买卖价差，同时承担库存和价格波动风险。

简化理解：

```text
买价 99.9
卖价 100.1
如果不断低买高卖，就可能赚取价差
```

但真实做市远不止这么简单。需要处理：

- 库存风险
- 单边行情风险
- 撤单和排队
- 盘口深度
- 延迟
- 手续费和返佣
- 极端行情

做市通常对工程、低延迟、风控和交易成本要求很高，不适合作为早期研究重点。

## 8. 多策略组合

单一策略很容易遇到失效期。多策略组合的目标是让不同收益来源互相补充。

常见组合方式：

- 趋势 + 均值回归
- 股票因子 + 债券配置
- 中低频策略 + 风险平价
- 不同市场、不同周期、不同资产的策略组合

关键不是简单叠加，而是检查：

- 策略之间相关性是否过高？
- 是否在同一类市场风险下同时亏损？
- 资金和杠杆如何分配？
- 单一策略失效时是否有暂停机制？

## 9. 策略评估指标

常见指标：

- 年化收益率
- 年化波动率
- 最大回撤
- 夏普比率
- 卡玛比率
- 胜率
- 盈亏比
- 换手率
- 单笔平均收益
- 策略容量

不要只看单一指标。一个策略可能年化收益很高，但回撤极大、换手率过高、容量很小，实盘价值未必高。

## 10. 常见误区

### 误区 1：把指标当策略

“RSI 低于 30 买入”只是规则，不一定是策略。还需要解释收益来源、风险、交易成本和失效条件。

### 误区 2：参数调到最好就是好策略

如果不断调参数直到历史曲线漂亮，很可能只是拟合了噪声。

### 误区 3：忽略市场环境

趋势策略在趋势市场表现可能很好，在震荡市场可能很差。策略要记录适用环境。

### 误区 4：收益来源不清楚

不知道为什么赚钱，就很难知道什么时候该停止。

## 11. 实践任务

1. 选一个趋势策略，用一句话写出策略假设。
2. 选一个均值回归策略，说明它可能在哪些市场环境下失效。
3. 找一个常见因子，写出数据来源、计算方法和可能风险。
4. 给任意策略补全：买入、卖出、仓位、止损、交易成本、失效条件。
5. 对比两个策略的相关性，思考是否适合放在同一个组合里。
