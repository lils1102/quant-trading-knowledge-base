# 资料调研

本文件记录知识库采用的资料来源、筛选原则和使用方式。目标是让后续文档可追溯，而不是只留下没有出处的结论。

## 1. 调研原则

本知识库优先采用以下资料：

1. **官方投资者教育资料**：用于解释股票、债券、ETF、订单类型、风险等基础概念。
2. **交易所和行业教育资料**：用于解释期货、期权、交易机制和产品结构。
3. **官方数据接口文档**：用于解释财报、宏观数据和数据获取方式。
4. **经典学术论文和数据集**：用于解释因子、组合理论、回测过拟合等研究问题。
5. **工程工具官方文档**：用于解释数据处理、时间序列验证和机器学习流程。

资料使用顺序：

```text
先查官方定义
  -> 再看行业实践
  -> 再看经典论文
  -> 最后转成可执行检查清单
```

## 2. 市场基础资料

### SEC Investor.gov

用途：解释股票、债券、订单类型、风险和资产配置等基础概念。

采用方式：

- 用于 `01-金融市场基础.md` 中股票、债券、订单和风险的定义。
- 用于 `08-投资组合.md` 中资产配置和风险分散的说明。

重点页面：

- Stocks
- Bonds or Fixed Income Products
- Types of Orders
- Asset Allocation
- Risk

### FINRA 投资者资料

用途：解释 ETF、基金、交易产品和投资者保护相关内容。

采用方式：

- 用于 ETF 结构、交易方式、风险点和产品差异说明。
- 用于提醒读者关注费用、流动性、折溢价和结构复杂度。

### Options Industry Council

用途：解释期权基础概念。

采用方式：

- 用于 `02-交易品种.md` 中看涨期权、看跌期权、行权价、到期日、隐含波动率等内容。
- 用于 `10-机器学习与AI量化.md` 中提醒期权数据特征和非线性风险。

### CME Group Education

用途：解释期货合约、保证金、到期、每日结算、套期保值和商品市场。

采用方式：

- 用于 `02-交易品种.md` 的期货章节。
- 用于 `07-风险管理.md` 中关于保证金、杠杆和强制平仓风险的说明。

## 4. 学术资料方向

### 组合理论

核心资料：Harry Markowitz 的 Portfolio Selection。

采用方式：

- 用于解释组合风险不仅来自单个资产波动，还来自资产之间的相关性。
- 用于 `08-投资组合.md` 中组合权重、相关性和风险分散的说明。

### 因子模型

核心资料：Fama-French 因子模型相关论文和数据。

采用方式：

- 用于说明价值、规模、市场、盈利能力、投资风格等因子思想。
- 用于解释因子研究为什么要做风险暴露和归因。

### 回测过拟合

核心资料：Bailey、López de Prado 等关于回测过拟合和 Deflated Sharpe Ratio 的研究。

采用方式：

- 用于 `05-回测系统.md` 中说明参数过度优化、多重检验和样本外验证的重要性。
- 用于 `10-机器学习与AI量化.md` 中说明模型验证和过拟合风险。

## 6. 资料转写规则

为了让文档易读，本知识库不会把资料原文大段搬运，而是按以下方式转写：

```text
官方定义
  -> 通俗解释
  -> 交易场景例子
  -> 常见误区
  -> 实践任务
```

例如，订单类型资料不会只列出定义，而会进一步解释：为什么市价单成交快但价格不确定，为什么限价单价格可控但可能无法成交，以及这些差异如何影响回测和实盘。
