# 阅读笔记模板

阅读论文、研报、策略文章或官方文档时，可以使用本模板记录关键信息。目标不是摘抄原文，而是把资料转化成可验证的研究问题。

## 1. 基本信息

```text
标题：
作者 / 机构：
发布时间：
资料类型：论文 / 研报 / 官方文档 / 数据文档 / 技术文档 / 其他
链接：
阅读日期：
```

## 2. 这份资料回答什么问题

用一两句话说明资料的核心问题。

```text
这份资料主要研究：

它试图回答：

这个问题为什么重要：
```

示例：

```text
这份资料研究价值因子是否能解释股票横截面收益。
它试图回答低估值股票是否长期获得更高风险补偿。
这个问题重要，因为它可以转化为选股和组合构建规则。
```

## 3. 核心假设

记录资料背后的假设。

```text
假设 1：
假设 2：
假设 3：
```

需要判断：

- 假设是否有经济含义？
- 假设是否能被数据检验？
- 假设是否可能已经被市场充分定价？
- 假设在哪些环境下可能失效？

## 4. 数据说明

```text
数据来源：
资产范围：
时间范围：
数据频率：日频 / 分钟 / Tick / 月度 / 季度 / 年度
是否包含退市资产：
是否考虑复权：
是否考虑公告时间：
是否说明数据清洗方法：
```

重点关注：

- 是否有幸存者偏差？
- 是否有未来函数？
- 是否有样本选择偏差？
- 是否说明交易成本？
- 是否说明缺失值处理？

## 5. 方法概括

用自己的话重写方法。

```text
步骤 1：
步骤 2：
步骤 3：
步骤 4：
```

避免只复制公式。需要确认自己能解释：

- 输入是什么？
- 输出是什么？
- 中间计算步骤是什么？
- 每一步是否能复现？

## 6. 主要结论

```text
结论 1：
结论 2：
结论 3：
```

每个结论后面加一句判断：

```text
这个结论是否依赖特定市场？
是否依赖特定时间段？
是否考虑交易成本？
是否经过样本外验证？
```

## 7. 风险和限制

记录资料没有充分解决的问题。

```text
限制 1：
限制 2：
限制 3：
```

常见限制：

- 样本时间太短。
- 只覆盖单一市场。
- 没有交易成本。
- 没有样本外测试。
- 没有说明数据发布时间。
- 没有考虑流动性。
- 结论可能来自多重检验。

## 8. 可以转化成什么研究任务

把资料转成可执行任务。

```text
任务 1：复现核心指标
任务 2：替换成本假设后重新测试
任务 3：换一个市场测试
任务 4：做样本外验证
任务 5：加入风控和组合约束
```

每个任务尽量写清楚输入和输出。

## 9. 需要的数据字段

```text
字段 1：
字段 2：
字段 3：
字段 4：
```

并记录：

```text
这些字段从哪里来？
什么时候可获得？
是否会被修正？
是否需要复权或标准化？
```

## 10. 复现计划

```text
第一步：获取数据
第二步：清洗数据
第三步：计算指标
第四步：做基础回测
第五步：加入交易成本
第六步：做样本外测试
第七步：记录结果和失败情况
```

## 11. 复现结果

```text
是否成功复现原结论：是 / 否 / 部分复现
差异在哪里：
可能原因：
是否继续研究：是 / 否
下一步：
```

```text
是否值得纳入知识库：是 / 否
是否值得转成策略研究：是 / 否
需要补充验证的地方：
```
